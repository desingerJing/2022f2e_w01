## 運行方式

1. clone 此專案到本地端

```
git clone https://github.com/desingerJing/2022f2e_w01.git
```

2. 專案環境建置 - 抓入需使用的 NPM

```
npm install
```

3. 啟動開發環境

```
npm run serve
```
